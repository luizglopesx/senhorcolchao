# Arquivos de Logo

Salve os arquivos da logo aqui:

| Arquivo | Descrição |
|---|---|
| `logo-principal.png` | Logo completa fundo branco — mascote + Sr. Colchão + Sleep Store |
| `logo-fundo-azul.png` | Versão para fundo azul marinho — texto amarelo |
| `logo-fundo-amarelo.png` | Versão para fundo amarelo — texto azul |
| `mascote.png` | Só o mascote (círculo amarelo), sem texto |

## Como usar

- **Fundo branco ou claro:** `logo-principal.png`
- **Fundo azul marinho (#001DA4) ou escuro:** `logo-fundo-azul.png`
- **Fundo amarelo (#F7CF00) ou azul claro:** `logo-fundo-amarelo.png`
- **Redes sociais / ícone:** `mascote.png`
